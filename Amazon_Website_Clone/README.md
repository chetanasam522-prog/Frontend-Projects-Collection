# Amazon Clone Website (HTML & CSS)
This project is a front-end Amazon website clone created using HTML and CSS.  
It focuses on replicating the layout and design of Amazon’s homepage and footer for learning and practice purposes.

-----

# Project Features
- Amazon-style navigation bar
- Product sections and layout
- Footer with multiple panels and links
- Amazon logo with transparent background
- Clean and responsive UI design

-----

# Technologies Used
- HTML5
- CSS3

-----

# How to Run the Project
- Download or clone the repository
- Open the project folder
- Double-click on index.html
- The project will open in your default web browser

-----

# Purpose of the Project
- To practice HTML and CSS fundamentals
- To understand website layout and styling
- To improve front-end development skills
