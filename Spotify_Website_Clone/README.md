## Spotify Web Player Clone

A responsive Spotify Web Player UI Clone built using HTML & CSS.  
This project replicates the basic layout and design of Spotify’s web interface including sidebar navigation, music cards, and music player controls.

---

## Features

-  Sidebar Navigation (Home, Search, Your Library)
-  Playlist & Podcast Sections
-  Music Cards (Trending, Popular Artists, Featured Charts)
-  Sticky Navigation Bar
-  Custom Styled Range Sliders (Progress & Volume Bar)
-  Responsive Layout
-  Google Fonts Integration (Montserrat)
-  Font Awesome Icons

---

## Technologies Used

- HTML5
- CSS3
- Font Awesome Icons
- Google Fonts
